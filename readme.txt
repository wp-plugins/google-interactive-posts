Enable Google+ interactive posts on your website in a few simple steps.

== Description ==

Interactive posts are one of the greatest features within Google+, but rarely utilised due to implementation complexity. The aim of this plugin is to help webmasters take the first steps towards implementation of interactive posts.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Follow further set-up instructions on the plugin page.

== Frequently Asked Questions ==

= What does this plugin do? =

It enables interactive sharing on Google+ from your posts by adding call to action buttons to shared posts.

= Is this plugin related to Google+ commenting? =

No.

= Why doesn't this plugin do anything when I install it? = 

Further configuration is required to get everything going. Please read and follow the instructions available from the plugin page or visit http://dejanseo.com.au/wordpress-plugin-google-interactive-posts/ for details.
